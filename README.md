# GPT Shaders

GPT Shaders is a lightweight shader pack for Minecraft Java Edition designed for Iris Shaders. It enhances Minecraft's visuals with a custom sky, custom sun and moon, animated water, warm lighting, and atmospheric effects while maintaining good performance.

## Features

- Custom sun
- Custom moon
- Enhanced night sky
- Improved daytime sky
- Animated water
- Warm torch and lava lighting
- Waving vegetation
- Atmospheric fog
- Performance-focused design

## Compatibility

- Minecraft Java Edition
- Iris Shaders
- NeoForge 1.21.1
- Fabric support planned

## Downloads

### Release Version

The Release version is intended for normal gameplay.

1. Download the latest release.
2. Place the ZIP file into your `.minecraft/shaderpacks` folder.
3. Enable GPT Shaders in Iris.

### Source Code

The Source Code is intended for developers and anyone who wants to learn how GPT Shaders works. You are free to explore and modify the shader source.
